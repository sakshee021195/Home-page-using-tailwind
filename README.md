# Home-page-using-tailwind
This project showcases a responsive navigation bar built using Tailwind CSS and JavaScript. The navbar adjusts dynamically for various screen sizes, providing a seamless user experience. A hamburger menu is implemented for smaller screens, which toggles the navigation links visibility on click.
